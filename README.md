# oneword-web

## 开发

1. 安装 pipenv

```shell
brew install pipenv
# brew upgrade pipenv  # 升级
```

2. 初始化应用和启动

```shell
cd work  # 切换到工具目录
git clone git@github.com:lessc/oneword_web.git
cd oneword_web

pipenv install
pipenv shell

bin/start
```
